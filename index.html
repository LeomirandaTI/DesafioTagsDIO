Tags h1-h6

Voltar

As tags h1, h2, h3, h4, h5 e h6 são utilizadas para criar titulos em uma página web. Sendo o h1 o título principal e o restante títulos secundários ou subtitulos.


Tag p

Voltar

A tag p é utilizada para criar um paragrafo na página web.


Tag mark

Voltar

A tag mark é utilizado para destacar uma palavra ou um trecho de um texto em uma página web. Pode-se pensar que é o equivalente a um marca texto.


Tag small

Voltar

A tag small é utilizada para diminuir o tamanha de um texto, como está sendo utilizado em todos os hyperlinks com a palavra ''Voltar''.


Tag i

Voltar

A tag i é utilizada para criar um texto em itálico.


Tag u

Voltar

A tag u é utilizada para criar um texto sublinhado.


Tag strong

Voltar

A tag strong é utilizada para criar um texto em negrito.


Tags de lista

Voltar

As tags ol, ul e li são utilizadas para criar listas, ordenadas e desordenadas, se utilizar a tag ol irá ser criado listas ordenadas, já a tag ul irá criar listas desordenadas e a tag li serve para criar os tópicos dentro da lista.


Tag a

Voltar

A tag a é utilizada para criar hyperlinks para visitar outras páginas web ou utilizar o recurso de ancoras, como está sendo utilizado em todos os hyperlinks com a palavra ''Voltar''.


Tag hr

Voltar

A tag hr cria uma linha uma linha horizontal na página web, como utilizado no inicio da página.


Tag sub e sup

Voltar

A tag sub serve para exibir um texto subscrito. Exemplo mais comum são em formulas químicas:"A fórmula química da água é H2O.". Ja a tag sup serve para exibir um texto sobrescrito, muito utilizado na matematica. Exemplo:"O cubo de 2 é 23."


Tag blockquote

Voltar

A tag blockquote é utilizada par criar um bloco de citação. Exemplo:

"Só existem dois dias no ano que nada pode ser feito. Um se chama ontem e o outro se chama amanhã, portanto hoje é o dia certo para amar, acreditar, fazer e principalmente viver. Viver é a coisa mais rara do mundo."
